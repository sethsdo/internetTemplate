Clients and Servers

Computers conected to the Web are called Clients and Servers.

  • Clients are parts of computer hardware or software that accesses a service made available by a server. The server is often (but not always) on another computer system, Web-accessing softwarein which case the client accesses the service by way of a network.
  • Servers are an instance of a computer program that stores webpages, sites, or apps. Less formally, any device that runs server software could be considered a server as well. Servers are used to manage network resources
